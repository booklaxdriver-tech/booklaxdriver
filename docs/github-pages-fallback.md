# GitHub Pages Static Fallback

This repository contains a static fallback copy of the BookLaxDriver website.

## Purpose

WordPress.com remains the official public website. GitHub Pages is only a backup/recovery preview path unless WordPress.com becomes blocked.

## Current Official Site

Use only:

- `https://booklaxdriver-ptljd.wordpress.com/`

Ignore:

- `https://booklaxdriver-itxey.wordpress.com/`

## Static Fallback Contents

- `index.html`
- `404.html`
- `robots.txt`
- `sitemap.xml`
- `.nojekyll`
- `.github/workflows/pages.yml`

## Required GitHub Setting

GitHub Pages may still require repository settings to be enabled manually:

Settings -> Pages -> Source: GitHub Actions

After that, the workflow can deploy the static fallback site.

## Important

Do not advertise the GitHub Pages URL as the official site unless WordPress.com/domain launch is blocked. The official goal remains `https://booklaxdriver.com` connected to the correct WordPress.com site.
